# Code_DSA

This repository is a Java-based DSA practice workspace. Each file focuses on a topic or problem set and keeps the implementation self-contained so you can study, tweak, and run examples independently.

## Contents

- `basic.java` - core fundamentals such as digit counting, gcd, reversing numbers, recursion, hashing, and sorting.
- `binarysearch.java` - binary search patterns, bounds, rotated array search, peak finding, and related variations.
- `binary_trees.java` - binary tree traversal and problem-solving practice.
- `binary_serach_tree.java` - binary search tree practice and standard BST operations.
- `bit_manipulation.java` - bitwise techniques and common bit manipulation problems.
- `greedy.java` - greedy strategy problems and implementations.
- `pattern.java` - pattern printing exercises.
- `recursion.java` - recursion drills and backtracking-style practice.
- `sliding_window_2_pointer.java` - sliding window and two-pointer techniques.
- `stacks_queue.java` - stack and queue based problems.
- `trie.java` - trie data structure operations and examples.

## How To Use

Compile any file directly with `javac`:

```bash
javac basic.java
java basic
```

Use the class name that matches the file you are running. Most files are independent practice modules, so you can compile and execute them one at a time.

## Notes

- The workspace is organized as a study notebook rather than a single application.
- File names are kept close to the DSA topic they cover, so you can jump to the area you want quickly.
